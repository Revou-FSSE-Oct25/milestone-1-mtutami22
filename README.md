# Personal Website - Mustika Tri Utami

## Overview
This is my personal portfolio website that showcases my skills, projects, and professional background as a Data Analyst and aspiring web developer. The website serves as a digital resume and platform for potential collaborators to learn about my work and get in touch.

## Features
- **Home**: Introduction and welcome section with personal photo
- **About Me**: Detailed biography, education background, and certifications
- **Projects**: Showcase of data analysis projects with descriptions and tools used
- **Collaborate**: Contact form for project inquiries and collaboration opportunities

## Technologies Used
- **HTML5**: Semantic markup for structure and accessibility
- **CSS3**: Custom styling for responsive design
- **GitHub Pages**: Free hosting platform
- **Git**: Version control
